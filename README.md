# Бот здорового питания (Production Version)

Этот бот помогает пользователям находить рецепты здорового питания, распределенные по категориям, с возможностью поиска и умного подбора.

## Основные возможности

- **Главное меню**: Быстрый доступ ко всем функциям.
- **Категории**: Просмотр рецептов по типам (Каши, Завтраки из яиц, Мясо и т.д.).
- **Поиск**: Поиск рецептов по названию, ингредиентам или тегам.
- **Умный подбор**: Рекомендации на основе времени и предпочтений (легкое/сытное/детское).
- **Админ-панель**: Добавление, редактирование и удаление рецептов через Telegram.
- **Советы**: Полезная информация о здоровом питании и питании детей.

## Технологии

- **Python 3.11+**
- **aiogram 3.x**
- **SQLite** (база данных рецептов)
- **FSM** (состояния для админ-панели и опросов)

## Структура проекта

- `main.py` — Точка входа.
- `bot/database/` — Работа с SQLite.
- `bot/handlers/` — Основная логика бота.
- `bot/admin/` — Логика панели администратора.
- `bot/keyboards/` — Inline-клавиатуры.

## Запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Убедитесь, что в `main.py` указан правильный токен.
3. Запустите бота:
   ```bash
   py main.py
   ```

## Админ-панель

Для доступа к админ-панели отправьте боту команду `/admin`. Не забудьте добавить свой User ID в список `ADMIN_IDS` в файле `bot/admin/handlers.py`.
